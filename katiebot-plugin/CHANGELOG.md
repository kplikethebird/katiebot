# Changelog

All notable changes to the KatieBot plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-27

### Added

- Initial release of KatieBot editorial coaching agent
- Core skill: `katiebot` - full editorial coaching with stage-aware feedback
- Skill: `eli5` - clarity checking for jargon, hand-waving, skipped steps
- Skill: `asshole-read` - hard feedback mode for stress-testing drafts
- Skill: `angle-finder` - generate five distinct angles for stuck pieces
- Slash commands for all four skills
- Writer model templates for per-project pattern tracking
- Style guide templates for codifying writer preferences
